# 依赖清单

本文档列出项目所需的依赖：**系统 / apt 安装的库**与 **随本仓库一并编译的 ROS2 接口包**。

## 系统要求

- **操作系统**：Ubuntu 22.04+（脚本 `install_dependencies.sh` 针对 Debian/Ubuntu）
- **架构**：x86_64 / amd64（常用）

## 构建工具

| 依赖项 | 版本要求 | 安装命令 |
|--------|---------|---------|
| CMake | >= 3.10 | `sudo apt install cmake` |
| GCC/G++ | >= 9 | `sudo apt install gcc g++` |
| Make | 最新 | `sudo apt install make` |
| pkg-config | 最新 | `sudo apt install pkg-config` |

## ROS2 依赖（apt / 官方源）

工作区编译与运行需要 ROS2 基础环境（以下以 **Humble** 为例，可按实际发行版替换路径与包名前缀）。

| 依赖项 | 说明 | 典型安装命令 |
|--------|------|----------------|
| ROS2 Humble（或 Jazzy 等） | 桌面版或精简版均可 | `sudo apt install ros-humble-desktop` |
| rclcpp | C++ 客户端库 | `sudo apt install ros-humble-rclcpp` |
| std_msgs | 标准消息 | `sudo apt install ros-humble-std-msgs` |
| rosidl_default_generators | 接口代码生成 | `sudo apt install ros-humble-rosidl-default-generators` |
| rosidl_default_runtime | 接口运行时 | `sudo apt install ros-humble-rosidl-default-runtime` |
| colcon | 工作区构建 | `sudo apt install python3-colcon-common-extensions` |
| rosdep | 依赖解析（可选） | `sudo apt install python3-rosdep` |

说明：**ament_index_cpp**、**builtin_interfaces** 等通常随 `rclcpp`/桌面元包安装，无需单独列出。

## 工作区内 ROS2 包（源码编译，非 apt）

本仓库在 `src/ros2/src/` 下包含下列包，需与 **`inspire_control_ros2`** 在同一工作区由 **colcon** 编译：

| 包名 | 路径（仓库内） | 说明 |
|------|----------------|------|
| **rh5dg2_interfaces** | `interfaces/RH5DG2` | RH5DG2（13 自由度）专用 `.msg` / `.srv` |
| **rh56f1_interfaces** | `interfaces/RH56F1` | RH56 系列（6 自由度）专用 `.msg` / `.srv` |
| **inspire_control_ros2** | `driver` | 节点可执行文件 `inspire_control_node`，依赖上述两个接口包 |

**声明关系**（见 `src/ros2/src/driver/package.xml`）：`inspire_control_ros2` **depend** `rh5dg2_interfaces`、`rh56f1_interfaces`、`rclcpp`、`std_msgs`。

推荐首次编译命令：

```bash
cd /path/to/serial_control/src/ros2
source /opt/ros/humble/setup.bash
colcon build --packages-select rh5dg2_interfaces rh56f1_interfaces inspire_control_ros2
source install/setup.bash
```

## 第三方库依赖（apt）

| 依赖项 | 版本要求 | 安装命令 | 用途 |
|--------|---------|---------|------|
| Boost (system) | >= 1.65 | `sudo apt install libboost-system-dev` | 异步串口（Asio） |
| Boost (thread) | >= 1.65 | `sudo apt install libboost-thread-dev` | 线程 |
| Boost (dev 元包) | >= 1.65 | `sudo apt install libboost-dev` | 头文件与 CMake |
| yaml-cpp | >= 0.6 | `sudo apt install libyaml-cpp-dev` | YAML 配置解析 |
| spdlog | >= 1.5 | `sudo apt install libspdlog-dev` | 日志 |

## 系统权限

| 权限项 | 说明 | 配置命令 |
|--------|------|---------|
| dialout 组 | 访问 `/dev/ttyUSB*` 等串口 | `sudo usermod -a -G dialout $USER` |

## 快速安装

### 方式1：使用安装脚本（推荐）

```bash
./install_dependencies.sh
```

脚本安装 **CMake/GCC/Boost/yaml-cpp/spdlog** 与串口组配置；**ROS2 若未安装**会打印官方安装指引。接口包与节点包需在安装 ROS2 后于工作区内 **colcon** 编译。

### 方式2：手动安装

参见 [README.md](README.md) 中「依赖安装」章节；第三方库命令与上表一致。

## 验证安装

```bash
cmake --version
gcc --version
pkg-config --modversion boost
pkg-config --modversion yaml-cpp
dpkg -l | grep spdlog

echo $ROS_DISTRO
ros2 pkg prefix rclcpp

groups | grep dialout
```

编译验证：

```bash
cd src/ros2 && source /opt/ros/humble/setup.bash
colcon build --packages-select rh5dg2_interfaces rh56f1_interfaces inspire_control_ros2
```

## 常见问题

### Q: 只编译 `inspire_control_ros2` 报错找不到 `rh5dg2_interfaces`？

先在同一工作区编译两个接口包，或使用：

`colcon build --packages-select rh5dg2_interfaces rh56f1_interfaces inspire_control_ros2`

### Q: 话题类型不是 `RegisterData`？

重构后消息定义在 **`rh5dg2_interfaces` / `rh56f1_interfaces`**，由 **`interfaces_profile`** 选型；请使用 `ros2 topic info`、`ros2 interface list -p` 核对。
